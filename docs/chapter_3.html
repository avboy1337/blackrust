在VSCode中配置Rust工程

VSCode插件推荐

搜索Rust，安装最上面两个即可：

第一个是官方的Rust，第二个是社区的rust-analyer，许多官方插件的问题在这里得到了解决。

创建Rust工程

Cargo是一个不错的构建工具，如果使VSCode与它相配合，那么这将会是一个十分便捷的开发环境。

cargo new HelloWorld

早期的 cargo 在创建项目时，必须添加 --bin 的参数，如下所示：

cargo new HelloWorld --bin

现在的版本，已经无需此参数，cargo 默认就创建 bin 类型的项目。

运行项目

cargo run

二进制文件在这里：

./target/debug/

快速检查代码是否存在错误

cargo check

定义项目依赖

Cargo.toml文件：

[dependencies]
rand = "0.3"
hammer = { version = "0.5.0"}
color = { git = "https://github.com/bjz/color-rs" }
geometry = { path = "crates/geometry" }

将依赖信息添加在这里即可。
